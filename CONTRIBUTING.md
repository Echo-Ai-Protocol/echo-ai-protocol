# Contributing to ECHO AI Protocol

ECHO is an open protocol.
Contributions are welcome when they improve clarity, correctness, or trust.

## What can be contributed

- Specification clarifications
- Schema improvements
- Examples
- Security analysis
- Research notes relevant to protocol assumptions

## What is out of scope

- Product features
- Centralized services
- Identity systems
- Token economics
- UI/UX layers

## Contribution process

1. Open an Issue describing the change
2. Discuss rationale and protocol impact
3. Submit a Pull Request
4. Changes are evaluated for:
   - protocol consistency
   - decentralization impact
   - trust implications
